 **Ravindra_Task1** is a simple Java console application in which the computer randomly generates a number between 1 and 100. The player attempts to guess the correct number by entering guesses through the keyboard. After each guess, the program provides feedback indicating whether the guessed number is too high or too low. The game continues until the correct number is guessed. It also keeps track of the total number of attempts made by the player and displays the result at the end. This project demonstrates the use of Java concepts such as loops, conditional statements, random number generation, user input, and variables. It is designed to improve logical thinking and problem-solving skills while providing a fun and interactive learning experience.
 
 **Ravindra_Task2** is a lightweight, console-based reservation management system developed in Java. It provides a simple and efficient way to handle customer bookings by performing core CRUD (Create, Read, Update, and Delete) operations through a command-line interface. The application utilizes a single-class architecture with a static inner class to manage data, ensuring portability and ease of execution. Users can dynamically add new reservations, view current records, and cancel existing bookings using unique identifier tags, making it an ideal tool for learning object-oriented programming principles and dynamic data collection management.
